Graph Plotter

Processing Library

Author: M. Umar Shahbaz

This is a GitHub repository containing a custom library for Processing IDE that draws graphs.

To use this library locally you first need to download Processing IDE. Then you can follow the instructions in the README.md of the GitHub repository to run it as you please. The library is highly customizable and easy to use, you only need to import the GP_unit.pde in your sketch.

Note: In the examples folder there is a sketch to draw a graph from a serial port called Serial graph.
